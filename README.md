# Tienda en línea - Proyecto Scrum

Este repositorio corresponde al proyecto de tienda en línea desarrollado en la materia **Proyecto II**.

## Contenido

- Product Backlog (PB-1 a PB-10)
- Sprint Schedule (Sprints 1, 2 y 3)
- Código fuente de la aplicación (se irá agregando conforme al desarrollo)

## Tecnologías previstas

- React + Vite
- Node.js (futuro backend)
- GitHub para control de versiones
- Monday.com para gestión de Sprints
